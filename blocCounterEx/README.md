# bloccounterex

Bloc 라이브러리를 사용하여 Flutter로 Counter를 만들어보자.

- Bloc 라이브러리를 사용법을 익힌다.
- BlocObserver로 상태 변화를 관찰한다.
- BlocProvider를 사용하여 자식에게 블록을 제공한다.
- BlocBuilder를 통해 새 상태에 대한 응답으로 위젯 빌드를 처리한다.
